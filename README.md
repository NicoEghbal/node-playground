# Node Playground

A collection of Node.js practice projects, coding exercises, and mini applications built while learning backend development.

## 📂 Projects

| Project | Description |
|---------|-------------|
| Shopping Cart CLI | A terminal-based shopping cart application. |
| More projects... | Coming soon |

## 🎯 Goals

- Practice JavaScript and Node.js fundamentals
- Build real-world terminal applications
- Improve problem-solving skills
- Learn clean code and project structure
- Prepare for backend development with Express.js and NestJS

## 🛠 Technologies

- JavaScript (ES6+)
- Node.js
- Readline
- CommonJS

## 🚀 How to Run

Clone the repository:

```bash
git clone https://github.com/your-username/node-playground.git
```

Navigate to a project folder:

```bash
cd 01-shopping-cart-cli
```

Run the project:

```bash
node app.js
```
